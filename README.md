# Meteor-deploy

```bash
$ curl -O https://raw.githubusercontent.com/websquared/meteor-deploy/master/centos/meteor-deploy-install.sh
```

```bash
$ meteor-deploy server
$ mkdir ~/my_cool_app
$ cd ~/my_cool_app
$ meteor-deploy config
$ meteor-deploy run
```
